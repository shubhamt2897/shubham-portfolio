Assistive Robot: Agentic Skill Library

Robotics Skill Library using NormaCore Hardware and Station API

A voice and chat - commanded assistive robot: an agent that perceives its environment through the robot's cameras, plans physical actions, and executes tasks autonomously while recovering from failures. Give it a goal in plain language and it figures out the steps; Claude doesn't just execute, it orchestrates. A full perceive → decide → act → verify pipeline, not a scripted demo.

The design is a modular agentic skill library: left unspecified, Claude chooses between a learned policy (SmolVLA, Pi0.x) and classical control (visual-servoing or IK with object pose and motion planning), or you pick a skill and Claude runs the orchestration.

  • 01
    Natural-language goal to actionSpeak or type a goal and the agent locates, picks, places, stacks, drags, and points.
  • 02
    Modular agentic skill libraryClaude selects between learned policies and classical control, or the user picks a skill directly.
  • 03
    Claude brain over MCP, voice optionalClaude Agent SDK drives the NormaCore Station API as an MCP server; optional STT→Claude→ElevenLabs voice loop. First autonomous pick-and-place on hardware.

Hierarchical Manufacturing Agent

PCB production scheduling · Gemini orchestration · vision-gated robot sorting

A full-stack MES-style agent for PCB manufacturing: Gemini 2.5 Flash reads live ERP orders from Arke, schedules the day's production queue by deadline and priority, and drives each phase through to physical robot execution with a human always in the loop.

  • 01
    Daily order scheduling with GeminiGemini reads Arke sales orders, resolves priority and deadline conflicts using EDF, and builds the day's production schedule across seven phases — all via natural language over Telegram.
  • 02
    VLA and vision gate accessed by GeminiGemini triggers the EfficientNet-B0 inspection gate and the SO-101 ACT policy. It also opens the live camera feed so defects, correct parts, and robot motion can be watched in real time.
  • 03
    Closed-loop defect handlingFailed inspection triggers reject trajectories and rework orders automatically; Gemini reports status and awaits operator approval before continuing.
